How to configure your sites

  • Copy the Sitecore site path from the Homepage Quick Info section
  • Enter your Site Url
  • If needed, add {lang} token in your "site URL" if you want to embed language at a specific place.
  • If you want to disable language embedding for a given site, click Advanced Mode and switch off embedding.
  • If you need to get a specific URL for a given language, click Advanced Mode and add Sitecore language in this field (e.g "en" or "fr-fr")
  • More help available at https://uquaisse.io
